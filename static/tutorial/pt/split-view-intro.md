# Visualização Dividida

O Zettlr oferece visualizações divididas para tornar seu trabalho mais produtivo. Nós fornecemos um layout de exemplo com três painéis de edição (um à esquerda e dois empilhados verticalmente à direita) para demonstrar essa funcionalidade. Você pode arrastar e soltar arquivos para criar novas divisões. Veja como funciona:

* Novos arquivos sempre serão abertos no último painel ativo (ou seja, aquele em que o cursor está)
* Você pode arrastar as abas de documento para fora da barra de abas para mover arquivos
* Mover abas para outras barras de abas ou para a área central de um editor moverá o arquivo para o painel correspondente
* Mover abas para o lado de qualquer painel de editor (topo, direita, fundo, esquerda) dividirá o painel nessa direção
* Fechar ou mover o último arquivo de um painel fará com que ele seja fechado automaticamente
* O Zettlr também suporta várias janelas. Pressione `Cmd/Ctrl+Shift+N` para abrir novas janelas
